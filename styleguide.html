📏 Espaçamento (Spacing Scale)

Padrão de espaçamento:

  • 8px - Micro espaçamento
  • 12px - Espaço dentro de elementos
  • 16px - Espaço padrão Entre elementos
  • 20-25px - Espaço entre cards
  • 30-40px - Espaço entre seções grandes
  • 60px - Padding de seção
  • 80px - Margin entre seções principais

📱 Breakpoints Responsivos

Dispositivo Largura Máxima Características
Desktop > 1200px Layout multi-coluna, menu horizontal
Tablet 768px - 1200px 2 colunas, menu ajustado
Mobile < 768px 1 coluna, menu toggle

✨ Animações e Transições

Duração padrão: 0.3s

Easing: ease ou ease-out

Exemplos:

  • Hover em botões: transform: translateY(-2px)
  • Hover em cards: transform: translateY(-5px) ou scale(1.05)
  • Entrada de elementos: fadeIn animação
  • Menu mobile: slideDown animação

✅ Boas Práticas

Para Manutenção do Design:

  • ✓ Sempre use as variáveis CSS definidas em :root
  • ✓ Mantenha a consistência de espaçamento (8px grid)
  • ✓ Teste mudanças em todos os breakpoints (desktop, tablet, mobile)
  • ✓ Use as classes CSS existentes ao adicionar novos elementos
  • ✓ Siga a hierarquia de cores: verde primário > azul > amarelo > tons claros
  • ✓ Adicione alt text em todas as imagens
  • ✓ Mantenha o contraste de cores para accessibility
  • ✓ Use semântica HTML apropriada (<section>, <article>, etc)
